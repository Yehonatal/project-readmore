# Project Readmore

**Readmore** is a book recommendation system where users can input the books they have read or found interesting and receive recommendations for other similar books. The app will offer a variety of features to enhance the user experience and provide personalized book suggestions.

## Stack Information

This is a [T3 Stack](https://create.t3.gg/) project bootstrapped with `create-t3-app`.
